Pyodide ADDRESS Test Harness

This test verifies that RexxJS can use the REQUIRE mechanism to load the Pyodide ADDRESS handler and execute Python code.

RexxJS Script with Pyodide

This test will execute the following RexxJS script:

/* Test Pyodide ADDRESS handler via REQUIRE mechanism */
SAY 'Starting Pyodide test...'

/* Test the resolution chain: Publisher registry -> Module registry -> Bundle */
REQUIRE "org.rexxjs/pyodide-address"

SAY 'Pyodide address handler loaded'

ADDRESS PYODIDE
SAY 'Switched to Pyodide address'

/* Test simple Python execution */
LET result = run code="print('Hello from Python!')"
SAY 'Python stdout:' result.output

/* Test Python variable assignment */
LET result2 = run code="x = 42; print(f'The answer is {x}')"
SAY 'Python variable stdout:' result2.output

SAY 'Test completed successfully!'
